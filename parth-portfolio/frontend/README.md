# Portfolio: Parth Joshi - 3D Artist & Motion Designer

Welcome to my personal portfolio! This project showcases my professional work in 3D modeling, hard-surface design, and motion design, built with a modern web experience.

## 🚀 Tech Stack

- **Framework**: [Angular 21](https://angular.dev/)
- **Animations**: [GSAP (GreenSock Animation Platform)](https://greensock.com/gsap/)
- **Icons**: [Lucide Angular](https://lucide.dev/guide/packages/lucide-angular)
- **Styling**: SCSS (Modern Glassmorphism & Responsive Design)
- **Deployment**: [Angular CLI](https://github.com/angular/angular-cli)

## 📐 Portfolio Structure & Layout

The webpage is designed as a smooth, single-page professional portfolio with the following sections:

### 🖱️ Custom Cursor
- **Dynamic Interaction**: A custom GSAP-powered cursor that morphs and scales when interacting with buttons, links, and cards, providing a premium feel.

### 🧭 Navigation Bar
- **Responsive Design**: A sleek, glassmorphic navbar that becomes sticky on scroll.
- **Quick Links**: Direct navigation to Hero, About, Projects, and Contact sections.

### ✨ Hero Section
- **Visual Impact**: Features a dynamic mesh background and high-impact typography.
- **Introduction**: A brief greeting introducing me as a 3D Modeler, Motion Designer, and Hard Surface Modeling artist.

### 👤 About Section
- **Professional Detail**: A deeper dive into my creative focus: Stylized Environments, Motion Design, and AR/VR optimization.
- ** GSAP Entrance**: Smooth scroll-triggered entrance animations for content and imagery.

### 💼 Experience Section
- **Career Highlights**: Detailed timeline of professional roles, including my current work at AR/VR Navigation.
- **Accomplishments**: Key highlights focusing on optimization, UV mapping, and real-time rendering.

### 🚀 Projects Section
- **Showcase**: Featured high-fidelity works like "High-Fidelity Headphones" and "Coffee Pour Animation."
- **Technical Specs**: Highlights specific tools (Blender) and categories for each project.

### 🛠️ Skills Matrix
- **3D & Motion**: Proficiency in Blender, ZBrush, and Hard Surface Modeling.
- **Technical Skills**: Expertise in Retopology, UV Unwrapping, Substance Texturing, and PBR Workflows.

### 🖼️ Portfolio Gallery
- **Expanded Assets**: A grid-based gallery featuring a variety of assets from "Mechanical Watches" to "Cyber Cars," showcasing versatility in scale and complexity.

### 📧 Contact Interface
- **Get In Touch**: A dedicated section for inquiries with a functional contact form and direct links to Email, Phone, and Location.

## 💻 Development

### Development server
To start a local development server, run:
```bash
ng serve
```
Navigate to `http://localhost:4200/`.

### Building
To build the project:
```bash
ng build
```
Artifacts will be stored in the `dist/` directory.

### Testing
- **Unit Tests**: `ng test` (Vitest)
- **E2E Tests**: `ng e2e`
