# Parth Joshi — Portfolio

Personal portfolio showcasing 3D modeling, hard-surface design, and motion design work.

## Structure

```
parth-portfolio/
├── frontend/    → Angular 21 app (port 4200)
├── backend/     → Express.js API (port 5000)
└── README.md
```

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
npm install
npm run dev
```

## Tech Stack

- **Frontend**: Angular 21, GSAP, SCSS
- **Backend**: Express.js, Mongoose, MongoDB Atlas
